Information

Description

An ACL (Access Control List) is a security mechanism that defines permissions and access rights for objects within the Active Directory structure.


Risk

Misconfigured ACL can create access points or privilege escalation that an attacker could use to compromise the domain.

: At least one domain admin as target
: At least one object has a path to domain admin
: At least one object admin of a computer
: Other


Point of Attention

Regularly review and clean up ACL entries for users and groups that no longer require them.
